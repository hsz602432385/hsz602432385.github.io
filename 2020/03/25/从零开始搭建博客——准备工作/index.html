从零开始搭建博客——准备工作

写在开始

最近偶然在QQ群里看到有一部分人名称后面带了一串网址，貌似域名还都是与自己名字相关的。出于好奇就点进去看了一眼，结果发现都是大家自己做的个人博客，里面放了一些自己平时学到东西后总结下来的笔记和经验。其实早在之前自己就有些想法做一个自己的博客，尤其是配上一个独属于自己的域名，感觉是一件很酷的事情。奈何懒，迟迟无法实现。正好这段时间因为疫情待在家，啥也不想干，索性就学点东西，自己动手坐下自己的博客，并将这个过程记录下来，分享一下。那么废话不多说，正式开始着手搭建自己的博客~

需要安装的各种包和软件

这一次搭建博客，我采用的是比较流行的GitHub搭配Hexo模式，GitHub不用多说，大家都懂。而Hexo一般没接触过博客这块内容的人可能听得比较少或者没听过，这是一种基于NodeJS的静态博客搭建框架，它支持MarkDown文本格式，可以实现个人文章的快速发布。同时，它的官网上有许多大神自己写好的主题，用户只需简单的配置即可进行主题间的切换，大大减少了用户开发博客前端的工作量，基本可以说是零开发。

这里，我列出了前期需要下载的环境和安装包：

  • Node.js
  • Git Bash

其实，主要需要配置好的就这两样，很简单，后面配置Hexo主要都是通过npm的方式进行。

接下来，详细讲述这两样东西的配置过程，并附上安装包链接。

购买一个域名（可选）

    之所以这块不强制是因为后面github配置好之后，可以直接使用xxxxx.github.io的方式访问博客，不过这样难免显得有点low哈哈哈哈，而且买一个域名也不贵，cn后缀第一年只要20几，com后缀第一年50几，所以不如少喝一杯奶茶，换来一个自己的域名。

    买域名好像一般比较普遍的是到Godaddy上面，但是这里我选择在阿里的万域网上买，也挺方便，网址是这个：万域网。在下面这个图片的方框中输入自己想要的域名，查询该域名是否已经被占用，如果没有的话，那么久可以选择自己需要的后缀进行购买。第一次买的话好像还需要在阿里云新建一个信息模板，上传一下个人信息之类的，也不麻烦，根据指示做就好了。

完成这一步之后，恭喜你，你就获得了一个自己的域名，之后的配置将在后面的教程中讲解。
