从零开始搭建博客——GitHub端博客搭建

写在开始

在上一节，我们做好了搭建博客需要做的准备工作，包括下载相关安装包、购买域名等等。在这一节，我们将正式开始博客的搭建。这篇文章主要讲解的是Github端博客的搭建，所以，在这之前要做的第一件事，当然就是注册一个Github账号！

创建Github仓库

首先，我们需要创建一个仓库，这个仓库的作用主要就是之后将博客的前端部分和写好的文章上传到这个位置保存，然后通过输入仓库的地址则可以进行访问。创建的具体流程如下：

​ 首先点击右上角“+”号，选择“New Repository”创建一个仓库。

​ 

打开之后，界面如下：

​ 这里需要输入创建仓库的相关信息，填写内容需要注意如下几点：

① Repository name : 这里仓库名称需要按照这个格式输入：用户名.github.io 比如，我的用户名为giser，那么我的仓库名就输入giser.github.io 这样子，之后通过输入这个仓库名，就可以访问到你的博客。

② Description ：这个随便输就可以

③ Public Or Private ：这里记得选择Public

④ README：这个可勾选可不勾选，勾选的话会自动生成一个ReadMe文件

​ 按照上面的要点输入好相关信息后，直接点击“Create Repository”就创建成功了。创建成功后，按理来说在浏览器输入username.github.io就可以访问了，但是刚创建的话不会立即生效，一般要等个十几二十分钟。
